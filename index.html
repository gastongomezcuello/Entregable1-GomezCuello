Simulador Comercial de Ventas

es una aplicación interactiva para gestionar ventas de productos. Permite registrar ventas de productos, visualizar un historial de ventas, calcular el monto total vendido y gestionar productos con facilidad. Los usuarios pueden añadir ventas mediante un selector de productos y una cantidad, mientras que el sistema guarda automáticamente los registros para mostrar un resumen actualizado.
